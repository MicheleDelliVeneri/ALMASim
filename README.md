# ALMASim

ALMASim is a FastAPI-based application for advanced querying and simulation of ALMA data.

## Features
- Query by targets
- Query by science filters
- RESTful API endpoints
- Dockerized deployment

## Requirements
- Python 3.12+
- Docker (optional for containerization)
